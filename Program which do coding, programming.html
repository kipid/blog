# Program which do coding/programming Let's teach a machine how to do coding a (JAVA) program of any kind. How to score?
  • Codes must be runnable without throwing an (compile/run time) error or exception.
  • Codes must be accurate enough to perform a job correctly.
  • Codes must be efficient and fast enough while doing a job.
  • It is good if codes are understandable to human programmers. (And possibly can be feedbacked by humans or other machines?)
  • It is good if codes are factorized/packaged so as to be reused (in the project, or later in other projects).
  • It is good if codes can use a existing packages.
